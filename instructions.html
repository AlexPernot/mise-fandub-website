Comment installer le fandub ?

promis c'est facile

1. Ouvrir le dossier du jeu

Dans Steam, clic droit sur le jeu, puis allez dans le menu Gérer, et choisissez Parcourir les fichiers locaux.

2. Extraire les fichiers de l'archive

Sélectionnez les fichiers et faites-les glisser dans le dossier du jeu.

3. Installer le fandub

Lancez le script installer fandub.bat en double-cliquant dessus. Ce script fait un backup des fichiers originaux et copie les nouveaux fichiers à leur place.

5. Régler les volumes dans le jeu

Pour une meilleure expérience, allez dans les options sonores dans le jeu, et réglez les niveaux comme sur l'image.

6. Optionnel : désinstaller le fandub

Si vous avez trouvé ça trop nul, vous pouvez lancer le script desinstaller fandub.bat en double-cliquant dessus pour retrouver le doublage anglais.
Si quelque chose s'est mal passé, allez dans les propriétés du jeu sur Steam, et cliquez sur vérifier l'intégrité des fichiers. Steam retéléchargera les fichiers modifiés.
