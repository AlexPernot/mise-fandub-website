Comment installer le fandub ?

promis c'est facile

1. Ouvrir le dossier du jeu

Dans Steam, clic droit sur le jeu, puis allez dans le menu Grer, et choisissez Parcourir les fichiers locaux.

2. Extraire les fichiers de l'archive

Slectionnez les fichiers et faites-les glisser dans le dossier du jeu.

3. Installer le fandub

Lancez le script installer fandub.bat en double-cliquant dessus. Ce script fait un backup des fichiers originaux et copie les nouveaux fichiers leur place.

5. Rgler les volumes dans le jeu

Pour une meilleure exprience, allez dans les options sonores dans le jeu, et rglez les niveaux comme sur l'image.

6. Optionnel : dsinstaller le fandub

Si vous avez trouv a trop nul, vous pouvez lancer le script desinstaller fandub.bat en double-cliquant dessus pour retrouver le doublage anglais.
Si quelque chose s'est mal pass, allez dans les proprits du jeu sur Steam, et cliquez sur vrifier l'intgrit des fichiers. Steam retlchargera les fichiers modifis.
